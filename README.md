# HealthCare Clinic (React + Vite + HashRouter)

Deploy-friendly template for a doctors/clinic website. Uses React Router **HashRouter** so you don't need any server rewrites.
TailwindCSS for styling, Framer Motion for subtle animations.

## Quick Start (Local)
```bash
npm i
npm run dev
```

## Deploy (Vercel)
1. Push this folder to a GitHub repo.
2. In Vercel → New Project → Import the repo.
3. Build command: `npm run build`
4. Output directory: `dist`
(With HashRouter, no `vercel.json` required.)

## Customize
- Edit content in `src/App.jsx` (services, doctors, address, etc.).
- Colors and spacing via Tailwind classes in JSX.
